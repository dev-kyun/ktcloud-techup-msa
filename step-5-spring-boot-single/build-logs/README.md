# build-logs

빌드/도커 로그 보관.

파일 설명
- `gradle-build-output.txt` bootJar 빌드 로그
- `bootrun-output.txt` bootRun 실행 로그
- `docker-build-output.txt` docker compose build 로그
- `docker-compose-output.txt` docker compose up/down 로그
- `docker-compose-ps.txt` docker compose 상태
- `curl-output.txt` curl 요청 결과
- `build-logs-summary.txt` 요약
- `build-logs-annotated.txt` 해설
