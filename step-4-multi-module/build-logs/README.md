# build-logs

Gradle 빌드/실행 로그 보관.

파일 설명
- `gradle-build-output.txt` build 로그
- `gradle-build-output.annotated.txt` build 해설
- `projects-output.txt` 모듈 목록 출력
- `projects-output.annotated.txt` projects 출력 해설
- `app-run-output.txt` app 실행 로그
- `app-run-output.annotated.txt` app 실행 해설
- `admin-run-output.txt` admin 실행 로그
- `admin-run-output.annotated.txt` admin 실행 해설
- `build-logs-summary.txt` 요약
- `build-logs-annotated.txt` 해설
